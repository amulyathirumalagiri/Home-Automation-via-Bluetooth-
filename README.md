## Project Name
Home Automation via Bluetooth

## Project Scope
The scope of this project is to develop a home automation system using Bluetooth technology. It enables users to wirelessly control home appliances through a mobile application, providing convenience, energy efficiency, and improved home management.

## Project Description
The Home Automation via Bluetooth project uses an Arduino Uno and an HC-05 Bluetooth module to control electrical appliances. Users can connect their smartphones to the Bluetooth module and send commands through a mobile application. The Arduino receives these commands and controls connected devices through a relay module, allowing appliances to be switched ON or OFF remotely.
